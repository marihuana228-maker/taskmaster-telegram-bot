# TaskMasterBot — Умный Telegram-бот для управления задачами

**Готовый Telegram-бот** с добавлением задач, дедлайнами и **автоматическими напоминаниями** за 1 час до срока.

### Что умеет бот:
- ➕ Добавление задач с дедлайном (в формате ГГГГ-ММ-ДД ЧЧ:ММ)
- 📋 Просмотр всех задач с статусом
- ✅ Отметка задач как выполненных
- 🗑 Удаление задач
- ⏰ **Автоматические напоминания** за 1 час до дедлайна
- Удобное меню с кнопками
- Хранение данных в SQLite (не теряются при перезапуске)

### Демо
Просто добавь бота в Telegram и напиши `/start`

### Как запустить самому (2 минуты):
1. `git clone https://github.com/marihuana228-maker/taskmaster-telegram-bot.git`
2. `cd taskmaster-telegram-bot`
3. `pip install -r requirements.txt`
4. Запусти `python bot.py`

Бот сразу готов к работе!

### Для бизнеса / личного использования
Отлично подходит для:
- Личного планирования
- Командного управления задачами
- Напоминаний о дедлайнах
- Простого трекера дел

Хочешь **доработку** под свои задачи (добавить категории, приоритеты, интеграцию с Google Calendar, рассылку и т.д.) — пиши, сделаю быстро и недорого.

**Технологии:** Python + aiogram 3 + SQLite + APScheduler
